Собираем проект:

$ bem make

Первая сборка с bem-tools займет некоторое время, так как именно этот момент в директорию проекта устанавливаются все необходимые npm-пакуты.

Для запуска сервера выполняем:

$ bem server

В результате вы увидите следующее сообщение:

info: Server is listining on port 8080. Point your browser to http://localhost:8080

Проблема?

Если порт 8080 уже используется другой программой, его можно переназнычить с помощью аргумента -p.
