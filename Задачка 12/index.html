The Path to Enlightenment

These instructions are for *nix platforms. We also have Windows instructions.

You can run the tests by calling the path.to.enlightenment.rb file.

In your terminal, while in the ruby_koans directory, type:

[ ruby_koans ] $ ruby path_to_enlightenment.rb

Red, Green, Refactor

In test-driven development (TDD) the mantra has always been red: write a failing test and run it, green: make the test pass, and refactor: look at the code and if you can make it any better
